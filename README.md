# 📱 集中式手機管理系統 (Centralized mobile phone management system)

> 一個集中式手機管理系統，使用 Web App 監控手機的放置狀態和環境狀態，並將資料記錄在 Google 試算表中，同時可利用 RFID 解鎖或 Web App 遠端、定時解鎖。

## 📖 專案簡介 (Introduction)

近年來，全球教育界為提升學生專注力，對於校園手機使用的規範日益嚴格。然而，傳統的手機保管箱僅具備物理存放功能，缺乏即時狀態監控、環境感測預警與數位化管理能力，若發生手機未歸位或意外狀況，管理端難以即時掌握。

基於此痛點，本專案開發了一套整合環境感測與雲端監控之集中式手機管理系統，採用 **ESP32 與 Mega2560 雙主控協作架構**，不僅實現了 20 個手機槽位的精準狀態監控，更結合了雲端 Web App，提供校園或特定高保密場所一套高效、模組化且易於維護的手機集中管理方案。

## ✨ 核心特色 (Key Features)

* **🧠 異質雙主控架構 (Dual-MCU Architecture)**
  由 Arduino Mega2560 負責 20 路微動開關的底層 I/O 實時輪詢，並交由搭載 FreeRTOS 的 ESP32 進行邊緣運算與雲端通訊，徹底解決單晶片 I/O 不足與聯網阻塞問題。
* **☁️ 可視化網頁監控 (Serverless Web Dashboard)**
  完全免架設實體伺服器！後端採用 Google Apps Script (GAS) 構建，狀態與歷史紀錄即時寫入 Google Sheets，並提供可視化的響應式 Web App，管理員可隨時隨地遠端監控手機放置情況和環境狀態並執行遠端解鎖或定時排程。
* **🔐 多元安全門禁 (Secure Access Control)**
  支援授權 RFID 卡片感應解鎖、Web App 遠端一鍵解鎖，以及自訂時間的自動排程解鎖，配合磁簧開關實現防呆關門自動上鎖。
* **🚨 主動式災防預警 (Active Emergency Response)**
  整合 MQ-135 氣體感測器即時偵測空氣品質與火災煙霧（如手機鋰電池自燃）；同時接入 **台灣中央氣象局 (CWA) 地震速報 API**。當遭遇異常煙霧或四級以上有感地震等危急情況時，系統將觸發蜂鳴器警報並強制自動解鎖，確保人員逃生時能及時取回手機，保障生命財產安全。
